# Local Development

### Clone Repo:

```shell
git clone https://github.com/michasa001/Rusty-Scarecrow-Composable-NFT.git
cd Rusty-Scarecrow-Composable-NFT
```

### Install dependencies:

```shell
yarn
```

### Run local network:

```shell
yarn chain
```

### Deploy contracts on a second terminal:

```shell
yarn deploy
```

### Run frontend on a third terminal:

```shell
yarn start
```
